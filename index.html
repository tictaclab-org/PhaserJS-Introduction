PhaserJS
Fiche 1

Pour ce module, on va se concentrer sur la création de jeux vidéos avec le langage JavaScript. Pour ce faire, on se servira de la librarie Phaser.

Commencer !

Avant d'entamer ce défi, assure toi d'avoir déjà fait ceux ci:

  • Ceci est la première fiche, pas d'autre fiche à compléter avant! :-P
Ensuite, assure-toi de te munir d'une grande motivation et soif d'apprendre!

Dans tout bon jeu vidéo, il y a un joueur. Et comme nous créeons notre jeu vidéo à partir de rien, on va aussi créer notre joueur.

Pour créer un joueur, on utilise pixilart.com. Une fois sur leur site, appuie sur "START DRAWING" (Anglais pour "COMMENCER À DESSINER").

Tu peux maintenant choisir la taile de ton personnage. Je te recommande fortement de choisir 40x40. Pour commencer ton dessin, fait alors "NEW DRAWING".

Maintenant, explore les outils disponibles à la gauche de ton écran. Dessine un joli personnage pour notre jeu!

Quand tu as fini, appuie sur Download, et séléctionne Download PNG. Ceci va télécharger ton personnage en format png.

Pour coder notre magnifique jeu, on va utiliser ReplIt. Navigue sur le lien et crée toi un compte si ce n'est pas encore fait. Sinon, connecte toi à ton compte.

Lorsque tu es connecté à ton (nouveau) compte, crée un nouveau Repl. Celui ci doit être dans le language "HTML, CSS, JS".

Une fois ce Repl créé, on doit importer la librarie Phaser. C'est cette librarie qui va nous permettre de programmer tout notre jeu, du début à la fin!

Pour l'importer, copie-colle cette ligne de code dans index.html, entre la ligne 9 et la ligne 10 (Donc juste en dessous du <body>)


              <script src="//cdn.jsdelivr.net/npm/phaser@3.14.0/dist/phaser.min.js"></script>

Voilà tout ce qu'on va changer dans le index.html pour maintenant. Attaquons nous au script.js!

Tout en haut du script.js, colle ceci:


              let config = {
                type: Phaser.AUTO,
                  width: 800,
                  height: 500,
                  physics: {
                      default: 'arcade',
                  },
                  scene: {
                    preload,
                    create,
                    update
                  }
              };

Ceci est simplement un objet de configuration pour notre jeu. Pour l'instant, on ne va pas trop y changer. Remarque juste qu'on utilise Phaser à la ligne 2, ensuite on définit la taille de notre écran (Tu peux changer ceci si tu veux), et après on ajoute une "physique".

Ensuite, on va utiliser cet objet de configuration. En ce faisant, on va créer notre jeu.


              const game = new Phaser.Game(config);

On initialise un nouveau jeu Phaser ici. Lance ton code et regarde si quelque chose change.

Toi aussi tu as une erreur? C'est normal! Il faut encore rajouter 3 fonctions après. Pour l'instant, ajoute les fonctions vides. On va les remplir après.


              function preload(){
              }

              function create(){
              }

              function update(){
              }

Si maintenant tu lances ton code, tu devrais voir un écran noir apparaître. Voilà ton jeu!

Preload en anglais veut dire "avant-charge". En fait, tout ce qu'on écrit dans cette fonction va s'executer avant que le jeu se charge. Par example, on devrait charger les images de nos personnages avant de charger le reste.

Aujourd'hui, on va juste charger notre personnage. D'abord, importe l'image que tu as créé plus tôt dans ton Repl. Regarde bien le nom, tu en aura besoin.

Ensuite, rajoute cette ligne de code dans la fonction preload. Remplace les XXXXXXXXXXX par le nom que tu veux donner à ton personnage, et YYYYYYYYY par le nom de ton image.


              this.load.image("XXXXXXXXXXX", "YYYYYYYYY.png");

Tu as maintenant chargé ton personnage. Si tu voudra rajouter d'autres personnages/objects/obstacles dans ton jeu plus tard, ce sera ici qu'il faudra le faire.

Create veut dire "Créer" en anglais. Ceci est la fonction dans laquelle on crée tout ce dont on aura besoin plus tard. Ceci veut dire, par example, de créer notre joueur, mais pas seulement.

Pour créer notre personnage, il nous faut se souvenir de comment on l'a appellé. Ensuite, on peut mettre la ligne de code suivante:


              this.player = this.physics.add.sprite(400, 360, "XXXXXXXXXXX");

PS: N'oublie pas qu'on travaille maintenant dans la fonction create. Aussi, essaie de changer les deux nombres (400 et 360) pour voir ce qu'ils font.

Dans cette fonction create on ne crée pas seulement le personnage. On va aussi créer les touches pour pouvoir bouger le personnage.

Une des touches dont on aura besoin est la touche D. En effet, la touche D fera bouger notre joueur à droite. Les autres touches dépendent de ton clavier, donc regarde bien avant de tout coder.

Pour ajouter une touche, fait comme ceci. Ici, je crée la touche D, mais il faudra faire ce code 4 fois, une fois par touche que tu veux utiliser.


              this.key_D = this.input.keyboard.addKey(Phaser.Input.Keyboard.KeyCodes.D);

Voilà, on a maintenant créé notre personnage et 4 touches. On est donc prêts à passer à la partie principale du jeu!

Update veut dire "Mettre à jour" en anglais. En effet, à l'opposition de preload et create, la fonction update va s'executer plusieurs fois et donc nous permettre bouger notre personnage.

Pour changer la vitesse du personnage, on fait comme suit. Essaie de mettre cette ligne dans ton code pour vérifier que tout marche bien. Souviens-toi qu'on travaille maintenant dans update.


              this.player.setVelocityX(100);

Le joueur devrait doucement partir à droite. Essaie les deux challenges suivants:

  • Fait avancer le joueur à droite plus rapidement
  • Fait avancer le joueur à gauche

Ce serait quand même chouette de pouvoir contrôler notre personnage. On va donc dire que lorsqu'on appuie sur la touche D, le personnage va à droite. Si elle n'appuie pas dessus, le joueur n'avance pas.

Il nous faut donc savoir comment vérifier si la touche D est pressée ou non. Heureusement pour nous, on a une façon simple de ce faire!


              this.key_W.isDown

On peut alors tout combiner. Ceci ressemblerait donc à ceci:


              if(/* Touche D appuyée */) {
                // Mettre vitesse X à 100
              } else {
                // Mettre vitesse X à 0
              }

Maintenant, lorsque tu appuies sur la touche D, ton personnage devrait avancer. Si tu n'appuies pas, il ne devrait rien faire.

Challenge: Étends ce code pour pouvoir bouger à gauche aussi!

Indice

Pour aller à gauche, c'est soit la touche A soit la touche Q. Regarde sur ton clavier avant d'écrire le code. Il va nous falloir faire quelque chose du style


                  if(/* Touche D appuyée */) {
                    // Mettre vitesse X à 100
                  } else if(/* Touche A/Q appuyée */) {
                    // Mettre vitesse X à -100
                  } else {
                    // Mettre vitesse X à 0
                  }

Réponse

Pour aller à gauche, c'est soit la touche A soit la touche Q. Regarde sur ton clavier avant d'écrire le code. Il va nous falloir faire quelque chose du style


                  if(this.key_D.isDown) {
                    this.player.setVelocityX(100);
                  } else if(this.key_A.isDown) {
                    this.player.setVelocityX(-100);
                  } else {
                    this.player.setVelocityX(0);
                  }

Excellent! Il ne reste plus qu'à ajouter les fonctionalités pour pouvoir monter et descendre. Sur mon clavier ce sont les touches W et S, mais ce sera peut être different sur le tien.

Tu peux essayer d'ajouter quelques fonctionalités en plus si tu as le temps!

  • Fait en sorte que ton personnage regarde à droite et à gauche dépendant de dans quelle direction il va. Utilise this.player.flipX = true; pour le retourner et this.player.flipX = false; pour le remettre droit.
